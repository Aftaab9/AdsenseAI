## Testing Instructions

### How to Test Each Image

1. **Start the server:**
   ```bash
   python start.py
   ```

2. **Open the web interface:**
   ```
   http://localhost:8000
   ```

3. **For each image:**
   - Upload the image from `static/examples/images/`
   - Use the suggested caption from this document
   - Select appropriate platform (Instagram recommended)
   - Set posting date if relevant (for festival images)
   - Click "Analyze Campaign"

4. **Compare results:**
   - Check if scores match expected ranges
   - Verify cultural alerts are detected
   - Confirm recommendation matches expected outcome
